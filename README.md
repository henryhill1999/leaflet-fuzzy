# leaflet-fuzzy

## Install

```bash
npm install --save leaflet-fuzzy
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'leaflet-fuzzy'
import 'leaflet-fuzzy/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
